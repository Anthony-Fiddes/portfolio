Projects

Fare-Comparison

Fare Comparison is an AppEngine application that I contributed to while working in a team of three developers during Google’s 2018 Computer Science Summer Institute in Atlanta. It served as a way to quickly and easily compare Lyft and Uber prices for a given ride. It utilized Google APIs to implement location autocomplete, and to return a map of the fastest route between the inputted source and destination locations. Python, HTML, CSS, and Javascript were all integral tools.

Unfortunately, as a result of a change in Lyft and Uber’s API policies, Fare Comparison no longer functions as intended.
